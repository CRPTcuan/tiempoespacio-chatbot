Ejemplo de Implementación del Chat Tiempoespacio

Este es un ejemplo de cómo implementar el chat en tu sitio web. El chat se abrirá automáticamente después de 11 segundos.

Código de Implementación:

<script src="chat-widget.min.js"></script>
<script>
    new TiempoespacioChat({
        position: 'bottom-right',  // Posición del botón (bottom-right, bottom-left, top-right, top-left)
        primaryColor: '#00C853',   // Color principal del chat
        autoOpen: true,           // Abrir automáticamente
        autoOpenDelay: 11000      // Tiempo de espera en milisegundos (11 segundos)
    });
</script>
